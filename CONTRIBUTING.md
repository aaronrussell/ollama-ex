# Contributing

Thanks for your interest in contributing to ollama-ex!

## Development Setup

1. Install dependencies:

   ```bash
   mix deps.get
   ```

2. Run the test suite:

   ```bash
   mix test
   ```

## Guidelines

- Add tests for new behavior.
- Keep changes focused and well-documented.
- Run `mix credo --strict` and `mix dialyzer` before submitting.

## Reporting Issues

If you find a bug, please include:

- Steps to reproduce
- Expected vs actual behavior
- Versions of Elixir and Ollama
